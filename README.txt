pasos para ejecutar prueba:

prerequisitos:

-- python 3.8.8
-- pipenv

PASOS:

1.- Crear entorno virtual con el comando "pipenv shell" al nivel del Pipfile
2.- ejecutar comando "pipenv install" para la instalación de dependencias
3.- ejecutar comando "python manage.py runserver 0.0.0.0:8000" para correr proyecto
    dentro de la carpeta eiyatest


Documentación:

 La documentación de la api se encuentra sobre el endpoint '/swagger/'
